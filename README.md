# Atrapa la Estrella - Python

## Descripción

Este proyecto corresponde a la actividad individual de Vibecoding Guiado: Desarrollo de una Aplicación o Mini-Juego con IA como Copilot.

El proyecto consiste en un mini-juego desarrollado en Python utilizando Tkinter. El jugador debe mover un cuadro azul con las flechas del teclado para atrapar una estrella. Cada vez que atrapa la estrella, suma puntos. El juego tiene un temporizador de 30 segundos.

## Tecnologías utilizadas

- Python
- Tkinter
- Visual Studio Code
- Git
- GitHub
- IA como apoyo de programación

## Archivos del proyecto

- `main.py`: contiene el código completo del mini-juego.
- `README.md`: descripción del proyecto e instrucciones.
- `Informe Vibecoding Atrapa la Estrella.pdf`: informe de la actividad.
